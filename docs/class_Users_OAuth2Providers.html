class Users \ OAuth2Providers

Provides an interface to supported OAuth2 providers. We need to provide a wrapper around this so we can map our oauth2 provider key (e.g. 'google' to the relevant ProviderInterface for OAuth2.) TODO this could be abstracted out into component-discovery for each provider
